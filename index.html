¿What is PocketMine-MP (PMMP)?

is an open source server software developed in PHP that allows users to create and customize their own Minecraft Bedrock Edition server, with highly configurable features.
It was launched on January 5, 2013 and supports platforms such as Windows, MacOS, Linux. It has an active community of developers and users who provide technical support and contribute to the development of
Because the plugins for this software are easy to code, it is possible to find a wide variety of these and there is an official and verified collection on poggit.
